EntityPrefabService - инфраструктурный сервис, который хранит глобальные игровые данные о
Entity префабах. Тк создавать Entity сущностей из обычных GameObjectPrefab мы не можем -
мы должны сначала "запечь" GameObjectPrefab в Entity сущность и далее использовать ее в качестве
префаба чтобы делать ее Instantiate в ECS-мир. Эти Entity-префабы не регулируются Addressabl'ами,
но у DOTS есть свои методы их загрузки/выгрузки.

Данный сервис - находит запеченный нами Singleton-Entity в Bootstrap сцене, в котором собраны все
запеченные ссылки на все возможные Entity-префабы в игре и может загружать нужные нам Entity-префабы,
выгружать ненужные или получить загруженный Entity-префаб из managed-системы. Для unmanaged-систем он также
кладет загруженные Entity-префабы в буффер на Singleton-Entity в Bootstrap сцене.
