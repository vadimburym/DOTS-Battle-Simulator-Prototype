SubSceneAwaiter - вспомогательный класс, к которому можно обратиться
чтобы подождать пока SubScene загрузится и будет готова к использованию.

SubScene - запеченная сцена для Entity - она имеет 2 состояния: открытое и закрытое (галочка справа от сцены)
открытое состояние - доступно только в редакторе. В билде будет только закрытое состояние.
SubScene в закрытом состоянии грузится несколько кадров после загрузки сцены, т.е.
Нужно дополнительно ждать загрузки SubScene прежде чем начинать инициализацию или тик.
